Projeto da Maratona Discovery da Rocketseat de gestão de finanças utilizando `HTML`, `CSS` e `Javascript`.

<br>

Tópicos estudados:
1. Formulários
2. Tabelas
3. Estilização
4. Displays, Positions
5. Responsividade
6. Manipulação do DOM
7. LocalStorage
8. Modal
